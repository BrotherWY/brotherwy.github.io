weex集成环境之安卓

Jul 25, 2017 | 技术 | 阅读

最近公司需要做一个toC的app，由于公司没有app开发人员，而且这个app主要就是一些数据展示的工作，所以boss决定用阿里的weex。这篇文章是我集成android环境走过的一些坑，记录一下！！

你需要安装如下环境:

  • java环境
  • android环境(科学上网)
  • node环境
  • 另外安装好weex-toolkit工具（全局安装）

初始化weex项目

weex init first cd first npm install

  • build: 源码打包，生成 JS Bundle
  • dev: webpack watch 模式，方便开发
  • serve: 开启静态服务器
  • debug: 调试模式

用扫码的方式开发挺烦的，网络访问比较慢，debug老是掉线，所以我选择连真机开发 下面就是集成安卓得到一些坑。
