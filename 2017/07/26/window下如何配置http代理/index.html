window下如何配置http代理

Jul 26, 2017 | 技术 | 阅读

因为一些android build要翻墙而且得配置代理，但是我配置socks代理没有用，很蛋疼，所以就转成了http代理，这篇文章是介绍下windows下如何socks转http。首先去弄一个影梭的账号(shadowsock)，如果只是上上网页，可以去试试搬瓦工。然后去下载privoxy。

privoxy

这个软件是可以把socks5 转换成http,下载安装后在config.txt的最后面加上:

forward-socks5 / 127.0.0.1:1080 .

android studio 配置代理

file->setting->搜proxy->http proxy

测试一下就ok了
